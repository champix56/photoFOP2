# photoFOP2

## Make your own album

this tool can generate album in :
- pdf
- rtf
- html
- xml

## options
- generate and save output XML files 
- manage cols and rows
- add your own XSL
- manage paper size (pdf/rtf)
- manage back image DPI
- add comment on each photo

## supported images
- pdf/rtf :
  - jpg
  - png
  - tif
  - bmp
  - svg
- html :
  - as pdf/rtf
  - all formats supported by <i>&lt;img&gt;</i> tag
